This is a simple Next.js project for visualization of the responses for

- **The problem 1**: [Three ways to sum to n](https://code-challenge-virid.vercel.app/utils/sum-to-n)
- **The problem 2**: [Fancy form](https://code-challenge-virid.vercel.app/swap)

## Working code to review

- `./app/utils/sum-to-n`: the page for the problem 1
- `./app/swap`: the page for the problem 2
- `./components`: React components
- `./hooks`: React hooks
- `./utils`: utility functions used by the entire app
